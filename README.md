# Training Planner

## Автор
Кирилл

## Описание
Training Planner — это приложение с графическим интерфейсом для планирования и учёта тренировок. Программа позволяет добавлять, просматривать, фильтровать и удалять записи о тренировках. Все данные автоматически сохраняются в JSON файл.

## Функциональные возможности
- Добавление тренировок с указанием даты, типа и длительности
- Проверка корректности ввода (формат даты, положительная длительность)
- Фильтрация записей по типу тренировки и дате
- Удаление выбранных тренировок
- Автоматическое сохранение и загрузка данных в формате JSON
- Удобный табличный просмотр всех записей

## Требования к системе
- Python 3.7 или выше
- Стандартные библиотеки Python:
  - tkinter (для графического интерфейса)
  - json (для работы с данными)
  - datetime (для работы с датами)
  - os (для работы с файлами)
